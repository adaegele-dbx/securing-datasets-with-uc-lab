# Solutions & Reference

Reference material for the *Securing Datasets with Unity Catalog* lab. You don't need these to
complete the lab — `lab_notebook.py` is self-contained. Use them if a step misbehaves or you want
the statements in one place.

| File | What's in it |
|------|--------------|
| `reference_sql.md` | Every SQL statement used in the lab (grants, mask/filter functions, ABAC policy, cleanup), copy-paste ready. |
| `abac_reference.md` | Deeper notes on ABAC — ordinary vs. governed tags, the production group-based patterns, and how this maps to enterprise workspaces. |

All objects live in `workspace.uc_security_lab` (Free Edition's catalog is always `workspace`).
